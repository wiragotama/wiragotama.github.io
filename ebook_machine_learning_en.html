Introduction to Machine Learning and Deep Learning (English Translation)

A free machine learning + deep learning e-book. Please download the book (as a whole) here.


This page is dedicated to my personal side project, which is writing an introduction to machine learning and deep learning book in Bahasa Indonesia (my mother tongue). The book aims to serve as an introductory-level reading for Indonesian undergrads. We believe you can understand this book easily since it is written in a semi-formal language. This book may not be suitable as a primary reference for studying machine learning since it contains only introductory-level materials (to serve as a primary reference, I believe a book should provide advanced-level materials as well). Even so, we hope this book can help you in its own way (e.g., as a quick introduction or additional material). It recommends you some great machine learning references, and I do recommend reading them after reading this book. The book is free, and you may download or share it for non-commercial purposes (but do not reupload!). Since this book isn't perfect, you might criticise, comment or suggest some improvements. Please feel free to email me.

I think it is natural to have some collaborators for a project of this scale.
